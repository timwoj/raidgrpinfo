raidgrpinfo
===========
Simple tool for looking at raid groups for World of Warcraft.

Dev setup:
==========
Python dependencies:
1. `virtualenv venv`
2. `source venv/bin/activate`
3. `mkdir lib`
4. `pip install -t lib -r requirements.txt`

Javascript dependencies:
`yarn install`

To build the app and run locally:
1. `webpack`
2. `dev_appserver.py`
